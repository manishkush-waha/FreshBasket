# Fresh Basket

## Introduction
Welcome to the Fresh Basket repository! This guide will help you clone the repository from GitHub and set up the project on your computer.

## Prerequisites
Before you begin, ensure you have the following installed on your computer:
- Git
- Node.js (if applicable)
- Any other dependencies specific to your project

## Cloning the Repository
Follow these steps to clone the repository:

1. Open your terminal or command prompt.
2. Navigate to the directory where you want to clone the repository.
    ```sh
    cd /path/to/your/directory
    ```
3. Clone the repository using the following command:
    ```sh
    git clone https://github.com/manishkush-waha/Fresh-Basket.git
    ```
4. Navigate into the cloned repository:
    ```sh
    cd Fresh-Basket
    ```

5. Install the necessary dependencies:
    ```sh
    npm install
    ```
    
6. Start Both Frontend and Backend server:
    ```sh
    npm run dev
    ```
